# Business Intelligence Project - Pharma Insights

## Dataset from: https://www.kaggle.com/code/tudorstroescu/pharma-sales-data-analysis-and-forecasting

## Features:
- Total Sales, Average Sales, Peak Sales, Growth Rate
- Drug Category filtering
- Daily/Weekly/Monthly filtering
- Sales Trend Analysis
- Sales Forecasting
- Price Impact Simulator
- Category Comparison
- Stacked Sales Overview
- Scatter Plot Analysis
- Multi-dimensional Comparison
- Category Correlation Matrix

## Stack:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
